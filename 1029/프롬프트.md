### 2. 다음에 나올 블럭이 무엇인지 표시해주는 창을 추가
이제 "다음 블록(Next Piece)" 창을 추가해서
다음에 떨어질 블록이 미리 보이도록 해보겠습니다.

✅ 변경 요약
HTML:
canvas id="next" 추가 (다음 블록 표시용 작은 캔버스)
UI에 “다음 블록” 제목 추가

CSS:
#next 캔버스 스타일 추가

JS:
nextPiece 변수 추가
spawnPiece() 수정 → piece = nextPiece; nextPiece = randomPiece();
renderNext() 함수 추가 → 작은 캔버스에 다음 블록 그림

💬 작동 원리 요약
기능	설명
nextPiece	다음에 등장할 블록 저장
spawnPiece()	현재 블록을 nextPiece로 교체, 그리고 새 nextPiece 생성
renderNext()	작은 next 캔버스에 다음 블록 그림
drawNextBlock()	1/2 크기로 블록을 그려서 맞춤

### 3. z키를 누르면 현재 블럭을 홀드 할 수 있는 기능을 추가.
-> Z 키를 누르면 현재 블록을 홀드한다.
  한 번 홀드하면, 해당 턴이 끝날 때까지 다시 홀드 불가 (공정성 유지).
  홀드된 블록은 왼쪽 상단 “HOLD” 박스에 표시된다.
  만약 이미 홀드된 블록이 있다면, Z를 눌렀을 때 현재 블록과 교체된다.

### 4. 배경음악 넣기.
-> 요즘 대부분의 브라우저(특히 Chrome, Edge, Safari)는 
  **“사용자가 직접 상호작용하기 전에는 오디오 재생을 금지”**합니다.
  즉, 페이지가 로드될 때 자동으로 audio.play()가 실행돼도 무음 처리됩니다.

  게임 시작 / 재시작 버튼을 눌렀을 때 배경음악을 시작하도록 바꿔주면 됩니다.
  즉, startGame() 안에서 음악을 재생하면 됩니다.
